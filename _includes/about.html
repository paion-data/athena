What is Athena

{{site.user_description}}

GraphQL

Athena makes it easy to build and maintain GraphQL web services for managing file metadata. Athena's API is clean and user-friendly, hiding the complexities of complicated data storage and query optimization from end-users

Big Data

Designed for Big Data and scalability, Athena has first-class support for OpenStack Swift and Hadoop HDFS as back-ends and flexible pipeline-style architecture which handles nearly any back-end for data storage

OpenStack

The long-term goal of Athena is to make it part of OpenStack ecosystem so to make it a quickly-provisioned and easily-managed SaaS infrastructural component
